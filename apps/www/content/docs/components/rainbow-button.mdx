<Step>Update the import paths to match your project setup.</Step>

<Step>Update `globals.css`</Step>

Add the following to your `globals.css` file:

```css title="app/globals.css" {2-6}
:root {
  --color-1: 0 100% 63%;
  --color-2: 270 100% 63%;
  --color-3: 210 100% 63%;
  --color-4: 195 100% 63%;
  --color-5: 90 100% 63%;
}
```

<Step>Add the required CSS animations</Step>

Add the following animations to your global CSS file.

```css title="app/globals.css" {2, 4-11}
@theme inline {
  --animate-rainbow: rainbow var(--speed, 2s) infinite linear;

  @keyframes rainbow {
    0% {
      background-position: 0%;
    }
    100% {
      background-position: 200%;
    }
  }
}
```


## Props

| Prop        | Type                                  | Default     | Description                                         |
| ----------- | ------------------------------------- | ----------- | --------------------------------------------------- |
| `children`  | `React.ReactNode`                     | `-`         | The content to be displayed inside the button.      |
| `className` | `string`                              | `-`         | Additional CSS classes to be applied to the button. |
| `variant`   | `"default" \| "outline"`              | `"default"` | The variant of the button.                          |
| `size`      | `"default" \| "sm" \| "lg" \| "icon"` | `"default"` | The size of the button.                             |
