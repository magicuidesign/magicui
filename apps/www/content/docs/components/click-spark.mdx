## Props

| Prop          | Type        | Default      | Description                                          |
| ------------- | ----------- | ------------ | ---------------------------------------------------- |
| `sparkColor`  | `string`    | `"#fff"`     | The color of the sparks.                             |
| `sparkSize`   | `number`    | `10`         | The size of the sparks.                              |
| `sparkRadius` | `number`    | `15`         | The radius of the spark explosion.                   |
| `sparkCount`  | `number`    | `8`          | The number of sparks per click.                      |
| `duration`    | `number`    | `400`        | The duration of the spark animation in milliseconds. |
| `easing`      | `string`    | `"ease-out"` | The easing function for the spark animation.         |
| `extraScale`  | `number`    | `1.0`        | Extra scale factor for the spark distance.           |
| `children`    | `ReactNode` | `undefined`  | The content to wrap.                                 |
| `className`   | `string`    | `undefined`  | Additional class names for the wrapper.              |
