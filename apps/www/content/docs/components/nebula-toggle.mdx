## Props

| Prop        | Type                         | Default     | Description                                             |
| ----------- | ---------------------------- | ----------- | ------------------------------------------------------- |
| `checked`   | `boolean`                    | `undefined` | Controlled checked state. If omitted, uses local state. |
| `onChange`  | `(checked: boolean) => void` | `undefined` | Callback fired when the state of the toggle changes.    |
| `className` | `string`                     | `undefined` | Custom CSS class styles to apply to the container.      |
