## Usage


```tsx showLineNumbers

```

## Props

| Prop        | Type          | Default | Description                                         |
| ----------- | ------------- | ------- | --------------------------------------------------- |
| `text`      | `string`      | `-`     | The text content to render with the kinetic effect. |
| `as`        | `ElementType` | `"h1"`  | The HTML element to render as.                      |
| `className` | `string`      | `-`     | Additional class names for the wrapper div.         |
