<Step>Add the required CSS into your global CSS file</Step>

```css title="app/globals.css" showLineNumbers
::view-transition-old(root),
::view-transition-new(root) {
  animation: none;
  mix-blend-mode: normal;
}

/* Keep the view-transition group on screen as long as the clip-path animation (the toggler sets --magicui-theme-toggle-vt-duration on each click). */
:root {
  --magicui-theme-toggle-vt-duration: 400ms;
}
::view-transition-group(root) {
  animation-duration: var(--magicui-theme-toggle-vt-duration);
}
```

## Usage

```tsx showLineNumbers
import { AnimatedThemeToggler } from "@/components/ui/animated-theme-toggler"
```

```tsx showLineNumbers
<AnimatedThemeToggler />
```

Use `animatedVariant` to change the clip-path shape of the view transition (circle, diamond, rectangle, hexagon, or star):

```tsx showLineNumbers
<AnimatedThemeToggler animatedVariant="star" />
```

## Props

| Prop               | Type                                                                                    | Default     | Description                                                                 |
| ------------------ | --------------------------------------------------------------------------------------- | ----------- | --------------------------------------------------------------------------- |
| `className`        | `String`                                                                                | —           | Additional classes to be added to the component                           |
| `duration`         | `Number`                                                                                | `400`       | Duration of the theme transition animation in milliseconds                  |
| `animatedVariant`  | `"circle" \| "diamond" \| "rectangle" \| "hexagon" \| "star"`                          | `"circle"`  | Shape used for the view-transition clip-path reveal                         |
