2. **Background Color on Flip:** Customize the background color when the button flips.

## Props

| Prop            | Type              | Default | Description                                                                                      |
| --------------- | ----------------- | ------- | ------------------------------------------------------------------------------------------------ |
| `fromTop`       | `boolean`         | `false` | Controls the flip direction. If true, the button flips from top to bottom.                       |
| `flippedText`   | `string`          | `-`     | The text displayed after the button flips. If not provided, it defaults to the same as children. |
| `flipBgColor`   | `string`          | `white` | The background color of the flipped side of the button.                                          |
| `flipTextColor` | `string`          | `black` | The color of the text displayed on the flipped side of the button.                               |
| `children`      | `React.ReactNode` | `-`     | The content to be displayed inside the button before flipping.                                   |
| `className`     | `string`          | `-`     | Additional CSS classes to customize the buttonâ€™s appearance.                                     |
