## Props

| Prop        | Type            | Description                                        | Default    |
| ----------- | --------------- | -------------------------------------------------- | ---------- |
| className   | string          | The class name to be applied to the component      |            |
| upperText   | string          | Upper half of the text to animate                  | "Separate" |
| lowerText   | string          | Lower half of the text to animate                  | "Away"     |
| duration    | number          | Duration until text is fully separated             | 1.5        |
