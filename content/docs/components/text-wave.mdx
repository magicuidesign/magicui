## Props

| Prop            | Type      | Default        | Description                                           |
| --------------- | --------- | -------------- | ----------------------------------------------------- |
| `text`          | `string`  | `""`           | The text to animate                                   |
| `waveFrequency` | `number?` | `0.5`          | The frequency of the wave animation (lower is slower) |
| `waveAmplitude` | `number?` | `20`           | The height of the wave in pixels                      |
| `color`         | `string?` | `currentColor` | The color of the text                                 |
| `className`     | `string?` | `""`           | Additional classes for the container                  |
