## Props

| Prop             | Type   | Description                                   | Default |
| ---------------- | ------ | --------------------------------------------- | ------- |
| className        | string | The class name to be applied to the component |         |
| text             | string | Text to be animated                           | ""      |
| default_velocity | number | Base scroll velocity of text                  | 5       |
