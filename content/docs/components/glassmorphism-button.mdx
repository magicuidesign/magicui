## Usage

```tsx

```

```tsx
<GlassmorphismButton>Glassmorphism Button</GlassmorphismButton>
```

## Props

| Prop        | Type                    | Default   | Description                                         |
| ----------- | ----------------------- | --------- | --------------------------------------------------- |
| `variant`   | `'circle' \| 'diamond'` | `diamond` | The visual style variant of the button.             |
| `color`     | `string`                | `purple`  | The color of the button.                            |
| `className` | `string`                | `-`       | Additional CSS classes to be applied to the button. |
| `children`  | `React.ReactNode`       | `-`       | The content to be displayed inside the button.      |
