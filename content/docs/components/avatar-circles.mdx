## Props

| Prop        | Type     | Default | Description                                   |
| ----------- | -------- | ------- | --------------------------------------------- |
| `className` | `string` | `-`     | The class name to be applied to the component |
| `numPeople` | `number` | `99`    | The number appearing in the last circle       |
