## Props

| Prop           | Type                           | Default                           | Description                                                                  |
| -------------- | ------------------------------ | --------------------------------- | ---------------------------------------------------------------------------- |
| `initialTags`  | `string[]`                     | `["python", "javascript", "c++"]` | An array of strings for the initial available tags.                          |
| `selectedTags` | `string[]`                     | `undefined`                       | A controlled state for the selected tags. Use this with the `onChange` prop. |
| `onChange`     | `(selected: string[]) => void` | `undefined`                       | Callback function that fires whenever the selection of tags changes.         |
| `className`    | `string`                       | `""`                              | Additional CSS classes to apply to the root component element.               |
