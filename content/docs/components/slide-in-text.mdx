## Props

| Prop        | Type     | Default | Description                                          |
| ----------- | -------- | ------- | ---------------------------------------------------- |
| `text`      | `String` | `-`     | The text need add slide animation                    |
| `className` | `String` | `-`     | The class name to be applied to text                 |
| `delay`     | `number` | `0`     | To add initial delay before the animation            |
| `duration`  | `number` | `0.5`   | Controls how long each child animation lasts         |
| `children`  | `number` | `0.1`   | Adds delay between the start of each child animation |
