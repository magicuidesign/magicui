## Props

| Prop               | Type              | Default  | Description                                                        |
| ------------------ | ----------------- | -------- | ------------------------------------------------------------------ |
| `children`         | `React.ReactNode` | Required | The content to be displayed inside the container.                  |
| `perspective`      | `number?`         | `1000`   | The CSS perspective value to simulate 3D depth.                    |
| `tiltMaxAngleX`    | `number?`         | `20`     | The maximum rotation angle on the X-axis.                          |
| `tiltMaxAngleY`    | `number?`         | `20`     | The maximum rotation angle on the Y-axis.                          |
| `scale`            | `number?`         | `1.01`   | The scale applied when the container is hovered.                   |
| `speed`            | `number?`         | `200`    | The speed of the spring animation (affects stiffness).             |
| `className`        | `string?`         | `""`     | Additional classes for the container.                              |
| `glareOpacity`     | `number?`         | `0.2`    | The opacity of the glare effect when hovered (set `0` to disable). |
| `rotationStrength` | `number?`         | `0.3`    | The strength for the rotation.                                     |
| `disabled`         | `boolean?`        | `false`  | Disables the tilt and hover effects.                               |
