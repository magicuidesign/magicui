## Props

| Prop        | Type     | Default | Description                                 |
| ----------- | -------- | ------- | ------------------------------------------- |
| `blobCount` | `number` | `8`     | Number of blobs to render                   |
| `blobSize`  | `number` | `250`   | Base size of the blobs (will vary slightly) |
| `range`     | `number` | `150`   | How far blobs can move from center          |
| `speed`     | `number` | `5000`  | How fast the colors change (ms)             |
| `blur`      | `number` | `60`    | Blur amount for the blobs                   |
