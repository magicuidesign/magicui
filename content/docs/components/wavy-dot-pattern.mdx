## Props

### Dot Pattern

| Prop       | Type   | Description                | Default |
| ---------- | ------ | -------------------------- | ------- |
| className  | string | Class name of the dot grid | -       |
| gridWidth  | number | Width of the dot grid      | 30      |
| gridHeight | number | Height of the dot grid     | 15      |
| dotWidth   | number | Width of the dot           | 8       |
| dotWidth   | number | Width of the dot           | 8       |
