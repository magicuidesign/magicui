## Props

| Prop          | Type                            | Description                                        | Default |
| ------------- | ------------------------------- | -------------------------------------------------- | ------- |
| className     | string                          | The class name to be applied to the component      |         |
| duration      | number                          | The class name to be applied to the shining border.| 14sec   |
| color         | `#${string}` | `#${string}`[]   | A string or string array to define border color.   | #ffffff |
| borderRadius  | number                          | Defines the radius of the border.                  | 8px     |
| children      | React.ReactNode                 | Contains react node elements.                      | {}      |
