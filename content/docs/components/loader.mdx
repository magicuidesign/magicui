## Usage

```tsx
import { Loader } from "@/components/magicui/loader";
```

```tsx
// This will display the loader centered over the entire screen
<Loader name = "leapfrog"/>

<div className="relative" ref={ref}>
  {/* This loader will only overlay this specific container */}
  <Loader name="leapfrog" containerRef={ref} />
</div>
```

## Props

| Prop              | Type                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      | Default       | Description                                                                                                        |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------- | ------------------------------------------------------------------------------------------------------------------ |
| `name`            | `'bouncy', 'bouncyArc', 'cardio', 'chaoticOrbit', 'dotPulse', 'dotSpinner', 'dotStream', 'dotWave', 'grid', 'hatch', 'helix', 'hourglass', 'infinity', 'jelly', 'jellyTriangle', 'leapfrog', 'lineSpinner', 'lineWobble', 'metronome', 'mirage', 'miyagi', 'momentum', 'newtonsCradle', 'orbit', 'ping', 'pinwheel', 'pulsar', 'quantum', 'reuleaux', 'ring', 'ring2', 'ripples', 'spiral', 'square', 'squircle', 'superballs', 'tailChase', 'tailspin', 'treadmill', 'trefoil', 'trio', 'waveform', 'wobble', 'zoomies'` | Required      | The loader style to render. Must be one of the predefined LDRS loader names.                                       |
| `size`            | `string`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  | `"40"`        | The size of loader.                                                                                                |
| `color`           | `string`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  | `"black"`     | The color of loader.                                                                                               |
| `speed`           | `string`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  | `"2"`         | The animation speed of the loader (must lie between `0.1` and `7.0`).                                               |
| `strokeLength`    | `string`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  | `-`           | The length of the stroke used by stroke-based loaders (if applicable must lie between `0.0` and `1.0`).            |
| `stroke`          | `string`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  | `-`           | The stroke thickness for stroke-based loaders (if applicable).                                                     |
| `bgColor`         | `string`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  | `bg-white/20` | The background color of the loading overlay (tailwind utilities classes for background color).                             |
| `loaderBgOpacity` | `string`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  | `-`           | The background opacity of the loader element (must lie between `0.0` and `1.0`).                                   |
| `blurBg`          | `boolean`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | `true`        | If `true`, applies a blurred semi-transparent background behind the loader; otherwise uses a flat overlay color.   |
| `containerRef`    | `HTMLElement`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             | `-`           | Reference to a container element. When provided, the loader overlays only that container instead of the full page. |
