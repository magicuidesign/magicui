## Props

| Prop        | Type            | Description                                        | Default |
| ----------- | --------------- | -------------------------------------------------- | ------- |
| className   | string          | The class name to be applied to the component      |         |
| duration    | number          | Duration to wait in between each char type.        | 200    |
| text        | string          | Text to animate                                    | ""      |
