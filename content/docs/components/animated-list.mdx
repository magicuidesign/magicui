## Usage

```tsx
import { AnimatedList } from "@/components/magicui/animated-list";
```

```tsx
<AnimatedList>
  <p>Item 1</p>
  <p>Item 2</p>
  <p>Item 3</p>
</AnimatedList>
```

## Props

### Animated List

| Prop        | Type     | Default | Description                       |
| ----------- | -------- | ------- | --------------------------------- |
| `className` | `string` | `-`     | The class name for the component  |
| `delay`     | `number` | `1000`  | The delay between each item in ms |
