Magic UI now supports Tailwind v4 and React 19 by default.

## How to use Tailwind v4

You can start by going to any component and following the instructions. Try [marquee](/components/marquee) for example.

## How to use Tailwind v3

If you're using tailwind v3, you can head over to [v3.magicui.design](https://v3.magicui.design) and follow the instructions there.

## How to upgrade your project to Tailwind v4

If you would like to upgrade your project to Tailwind v4, you can follow the steps here [https://ui.shadcn.com/docs/tailwind-v4](https://ui.shadcn.com/docs/tailwind-v4).
